## How it renders

Markdown is converted to HTML by the device's web server (via MD4C). The web
viewer shows that HTML directly; the LCD viewer parses a safe subset of it into
the on-screen layout.

1. fetch
2. convert
3. render

> Block quotes, lists, rules, and `inline code` are all supported.
